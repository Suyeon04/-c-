# 등급 처리 프로그램


<br>Built With
--------------------
<ul>
  <li>Mysql</li>
  <li>c++</li>
</ul>

<br>동아리 소개
--------------------
<li>지도교사 : 백현정 선생님</li><br>

<ul>
  <li>회원 가입(로그인)</li>
  <li>중간/기말/수행평가 평가 비율 기록</li>
  <li>중간/기말/수행평가 점수 기록</li>
  <li>고사 점수 및 총 점수 확인</li>
</ul>
